# 自動ラベル付与・カレンダー登録ツール
# セットアップマニュアル

このシステムは、Google Apps Script (GAS) 環境で自身の未読就活メールを監視・取得し、Gemini APIを用いて締め切り日を自動解析の上、曜日付きの期限ラベル（例：「06/25(木) まで」）および「処理済み」ラベルを自動で付与し、さらに「就活」カレンダー（別カレンダー）に終日イベントとして期限を自動登録するものです。

---

## 🛠️ 事前準備手順

本システムの構築には、**Gemini APIキーの取得**と、**GAS側での環境変数設定**が必要です。

### STEP 1. Gemini APIキーの取得
1. [Google AI Studio](https://aistudio.google.com/) にアクセスし、APIキー（GEMINI_API_KEY）を取得します。

---

### STEP 2. GAS (Google Apps Script) の設定

1. [Google Apps Script](https://script.google.com/) で新規プロジェクトを作成し、`code.js` のコードをエディタ（`コード.gs`）に貼り付けて保存します。
2. **スクリプトプロパティの追加**:
   - 左メニューの歯車アイコン（プロジェクトの設定）をクリックします。
   - 「プロジェクトの設定」画面の下部にある「スクリプトプロパティを追加」をクリックし、以下の環境変数を設定して保存します。
      - **`GEMINI_API_KEY`**: 取得した Gemini API キー
      - **`CALENDAR_NAME`**: （任意）登録対象のGoogleカレンダー名（デフォルトは「就活」）。指定された名前のカレンダーが存在しない場合は、実行時に自動で新規作成されます。

---

## 🔑 初回実行と権限の承認

1. GASのエディタ画面に戻ります。
2. 実行する関数として **`main`** を選択し、「実行」をクリックします。
3. 初回実行時（またはカレンダー連携機能追加後の初回実行時）、Googleアカウントへのアクセス権限（GmailおよびGoogleカレンダーへのアクセス）の承認を求めるダイアログが表示されます。
4. 画面の指示に従ってアクセスを許可（認可）します。
   - ※「このアプリは Google で確認されていません」という警告画面が出た場合は、「詳細」 > 「安全ではないページに移動」をクリックして進めてください。
5. 実行ログに「処理対象の未読メールはありません。」または処理件数が表示されれば、正常に動作しています。

---

## ⏰ 定期トリガーの設定

1時間に1回自動で処理が実行されるよう、GASのトリガーを設定します。

1. GAS画面の左メニューにある時計アイコン（トリガー）をクリックします。
2. 右下の「**トリガーを追加**」をクリックします。
3. 以下のように設定して「保存」します：
   - 実行する関数を選択: `main`
   - デプロイ時に実行する選択: `Head`
   - イベントのソースを選択: `時間主導型`
   - 時間ベースのトリガーのタイプを選択: `時間ベースのタイマー`
   - 時間の間隔を選択（時間）: `1時間おき`
   - エラー通知設定: `毎日まとめて通知`（お好みで変更可）

---

## 🧹 古い期限ラベルの自動削除（クリーンアップ）

本システムは、期限が過ぎて不要になったラベルがGmail内に増え続けるのを防ぐため、自動クリーンアップ機能を備えています。
- **仕様**: 期限から**1週間（7日）**経過すると、その期限ラベル（「MM/DD(曜日) まで」形式のラベル）は自動的に削除（クリーンアップ）されます。
- **設定変更**: 削除されるまでの日数を変更したい場合は、`code.js` の `main` 関数内にある `cleanupOldLabels(7);` の引数（日数）を書き換えてください。

---

## ⚙️ カスタマイズ（検索条件とラベル色）

このスクリプトは、ユーザーご自身の環境に合わせて「検索クエリ」や「曜日ごとのラベルカラー」を任意に変更することができます。
変更する場合は、`code.js` 内の以下の該当箇所を直接書き換えてください。

### 1. 検索クエリの変更（code.js: 16行目付近）

取得対象となるメールの条件（Gmailの検索ボックスに入力するクエリと同じ構文）を変更できます。

```javascript
// 【カスタマイズ可能】検索クエリ（件名・本文問わず、関連キーワードが含まれていればGeminiに判定させる）
// 自身の環境に合わせて「"企業名"」を追加するなど、自由に変更可能です。
const SEARCH_QUERY = 'is:unread -label:処理済み ("締め切り" OR "締切" OR "〆切" OR "期限" OR "提出" OR "予約" OR "受験" OR "受検" OR "テスト" OR "エントリーシート" OR "ES" OR "インターン" OR "選考") -("メルマガ" OR "ニュースレター" OR "コラム" OR "マガジン")';
```
ご自身の大学名や企業名を `OR "キーワード"` のように追加したり、不要なキーワードを削ることで、自分好みにカスタマイズが可能です。

### 2. 曜日ごとのラベル色の変更（code.js: 280行目付近）

新しく作成される曜日ごとの期限ラベルの色を変更できます。
※注意: Gmail APIの仕様上、**決められた特定のカラーパレットの色のみ**許可されています。

```javascript
// 【カスタマイズ可能】曜日ごとのラベル色設定
// Gmail APIがサポートしている公式カラーパレットから設定
// 許可されているカラーコード（例: #a479e2, #fb4c2f, #4a86e8, #16a766, #fad165, #cccccc など）自由に色を割り当てられます。
const colors = {
  '月': { backgroundColor: '#a479e2', textColor: '#ffffff' }, // 紫
  '火': { backgroundColor: '#fb4c2f', textColor: '#ffffff' }, // 赤
  '水': { backgroundColor: '#4a86e8', textColor: '#ffffff' }, // 青
  // ...
};
```
背景色（`backgroundColor`）に利用可能な代表的なカラーは以下の通りです：
- 紫: `#a479e2`, `#8e63ce`
- 赤: `#fb4c2f`, `#e66550`
- 青: `#4a86e8`, `#a4c2f4`
- 緑: `#16a766`, `#43d692`
- 黄: `#fad165`, `#f2c960`
- 灰: `#cccccc`, `#999999`
